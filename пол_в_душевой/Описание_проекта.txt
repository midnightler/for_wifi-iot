
Термостат для управлениея тёплым полом в душевой.
 Прошивка из проекта   wifi-iot.com   

При сборке добавить 5 настроек (Вр.ВКЛ., Вр.ВЫКЛ., Темп.Мин., Темп.Макс., Режим)
 (или назовите на Ваше усмотрение),переменных 0
 Страница настроек - TermSet (или на Ваше усмотрение).
 При обрыве и других проблемах с датчиками, работает по сохраненной температуре.
 Ограничил время такой работы десятью минутами.
 Кнопкой "принять" сохраняются настройки введённые на главной,
 Кнопка "режим" меняет режим по кругу:
 Выкл > работа по времени > работа постоянно.
 Внимание!!! Кнопки на главной не работают при опции "Динамический веб".!!
 
 Обязательные опции:

  GPIO
 Настроить входы и выходы.
 Выход - GPIO4,
 кнопка - GPIO0
 инд. кр. - GPIO12
 инд. зел. - GPIO15
 
 датчик - DS18B20
 
 редактор майнпейдж-2
  В редактор  "майн пейдж" можно добавить:
 <pre>DS18 Температура  <font size="7" color="crimson">_DSW1_</font> </pre>
 для отображения температуры. Или свой.
 
 Конструктор кода 
 Выход - строка - 12.
 Кнопка - строка - 13.
 Датчик - строка - 97.
 Индикаторы - строки - 6, 7,

 Для NOOS убрать строку:  vTaskDelay(1000 / portTICK_PERIOD_MS); 
sensors_param.cfgdes[0] - время вкл. термостата
sensors_param.cfgdes[1] - время выкл. термостата
sensors_param.cfgdes[2] - темп. включения
sensors_param.cfgdes[3] - темп. выключения
sensors_param.cfgdes[4] - mode режим работы
